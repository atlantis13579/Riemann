# Introduction

Riemann is a 3D Geometry / Collision Detection / Physics Engine with zero dependencies

## Features

Rigid Body Dynamics Simulation

Position Based Dynamics Simulation

* Simulation and Collision Detection for various shapes
    * Sphere
    * Oriented Box
    * Capsule
    * Convex Mesh
    * Triangle Mesh
    * Height Field

* Collision detection
    * Raycast Test
    * Overlap Test
